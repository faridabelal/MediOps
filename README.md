# MediOps

MediOps is a hospital workforce management application designed to help healthcare teams coordinate staffing, monitor operations, and manage employee schedules from a centralized dashboard.

Built as a React application, MediOps simulates common hospital workforce management workflows including staffing requests, department management, employee scheduling, and operational reporting.

---

## Features

- Workforce operations dashboard with real-time staffing overview
- Employee and department management
- Shift scheduling and availability tracking
- Coverage request management
- Operations queue for staffing shortages
- Reporting dashboard with workforce metrics
- Simulated hospital data for realistic workflows
- Role-based interface for administrators and employees
- Protected routes using authentication

---

## Technologies

- React
- JavaScript (ES6+)
- React Router
- Context API
- Vite
- CSS

## Getting Started

### Clone the repository

```bash
git clone https://github.com/faridabelal/mediops.git
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```

---

## Project Purpose

MediOps was developed to explore how software can improve hospital workforce coordination by providing a centralized platform for staffing, scheduling, and operational decision-making.

The application demonstrates frontend architecture, state management with React Context API, protected routing, reusable components, and responsive interface design while modeling real-world healthcare operations.
